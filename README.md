# MAITRI – Multimodal AI Assistant for Psychological & Physical Well-Being Monitoring

MAITRI is a cutting-edge multimodal AI assistant designed to monitor and support psychological and physical well-being through real-time emotion analysis and speech pattern recognition.

## Architecture

- **Backend**: FastAPI with Python, integrating computer vision and audio processing.
- **Frontend**: React.js with Tailwind CSS and Lucide icons.
- **Models**: State-of-the-art deep learning models for emotion detection (Face & Speech).
- **Fusion Engine**: Intelligent merging of multimodal inputs for accurate stress and well-being assessment.

## Getting Started

### Backend Setup
1. `cd backend`
2. `pip install -r requirements.txt`
3. `python main.py`

### Frontend Setup
1. `cd frontend`
2. `npm install`
3. `npm start`
